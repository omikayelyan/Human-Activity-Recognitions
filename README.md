# Human Activity Recognition 

## Project Overview

This project compares multiple machine learning classification algorithms 
on the Human Activity Recognition (HAR) dataset.

The objective is to evaluate how different models perform in recognizing 
human activities based on smartphone sensor signals.

The project includes:
- Data loading and preprocessing
- Optional data augmentation
- Training multiple ML classifiers
- Performance evaluation using multiple metrics
- Confusion matrix visualization
